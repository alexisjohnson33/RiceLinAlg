\beginedxvertical{Image Definition}

\beginedxtext{Image Definition and Relation with Span}

{\keya{\bf{Definition.}}}  
Given a linear transformation $T: V\rightarrow W$, the  {\keyb{\bf image}} of $T$ is the set
\[ \mathrm{Image}(T) =  \left\{ T(v) \ : \ v \in V\right\}. \] 

{\keya{\bf{Proposition.}}}  
If $T: \R^n \rightarrow \R^m$ is a linear transformation with standard matrix $A$, then the image of $T$
is the span of the columns of $A$.  (This is sometimes also called the column space of $A$.)  

\endedxtext

\beginedxvertical{Into Definition and Exercise}


\beginedxtext{Into Definition}

{\keya{\bf{Definition.}}}  
A linear transformation $T: V\rightarrow W$, is  {\keyb{\bf into}} if, for every $u, v\in V$ with $u\ne v$,
we have $T(u) \ne T(v)$.  

\endedxtext

\beginedxvertical{Kernel Definition and Questions}


\beginedxtext{Kernel Definition}

{\keya{\bf{Definition.}}}  
Given a linear transformation $T: V\rightarrow W$, the  {\keyb{\bf kernel}} of $T$ is the set
\[ \mathrm{Ker}(T) =  \left\{ v\in V \ : T(v) = \veco\}. \] 

{\keya{\bf{Proposition.}}}  
A linear transformation $T: V \rightarrow W$ is into if and only if $\mathrm{Ker}(T) = \{ \veco \}$.  

\endedxtext

\beginedxtext{Into Proposition}


{\keya{\bf{Proposition.}}}  
If the linear transformation $T: \R^n \rightarrow \R^m$ has standard matrix $A$, then $T$ is into if and only if $A$ has no free variables.  It is impossible for $T$ to be into if $n>m$.
