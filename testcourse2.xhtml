Definition. Given a linear transformation [mathjaxinline]T: V\rightarrow W[/mathjaxinline], the kernel of [mathjaxinline]T[/mathjaxinline] is the set

[mathjax]\mathrm{Ker}(T) = \left\{ v\in V \ : T(v) = \mathbf{0}\} .[/mathjax]

Proposition. A linear transformation [mathjaxinline]T: V \rightarrow W[/mathjaxinline] is into if and only if [mathjaxinline]\mathrm{Ker}(T) = \{ \mathbf{0}\}[/mathjaxinline].
