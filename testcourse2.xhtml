Definition. If [mathjaxinline]T: V \rightarrow W[/mathjaxinline] and [mathjaxinline]S: W \rightarrow V[/mathjaxinline] are linear transformations, we say that [mathjaxinline]S[/mathjaxinline] and [mathjaxinline]T[/mathjaxinline] are inverses of one another if [mathjaxinline]S(T(v)) = v[/mathjaxinline] for all [mathjaxinline]v\in V[/mathjaxinline], and [mathjaxinline]T(S(w)) = w[/mathjaxinline] for all [mathjaxinline]w\in W[/mathjaxinline]. If this is the case, we can write [mathjaxinline]S = T^{-1}[/mathjaxinline] and [mathjaxinline]T = S^{-1}[/mathjaxinline].

If [mathjaxinline]S = T^{-1}[/mathjaxinline] exists we say that [mathjaxinline]T[/mathjaxinline] is invertible.

Proposition. A linear transformation [mathjaxinline]T[/mathjaxinline] is invertible if and only if it is both into and onto.

Definition. If [mathjaxinline]A[/mathjaxinline] is an [mathjaxinline]m\times n[/mathjaxinline] matrix and [mathjaxinline]B[/mathjaxinline] is an [mathjaxinline]n\times m[/mathjaxinline] matrix, we say that [mathjaxinline]A[/mathjaxinline] and [mathjaxinline]B[/mathjaxinline] are inverses of one another if [mathjaxinline]AB = I_ m[/mathjaxinline] and [mathjaxinline]BA = I_ n[/mathjaxinline]. If this is the case, we can write [mathjaxinline]B = A^{-1}[/mathjaxinline] and [mathjaxinline]B = A^{-1}[/mathjaxinline].

If [mathjaxinline]B = A^{-1}[/mathjaxinline] exists we say that [mathjaxinline]A[/mathjaxinline] is invertible.

We've noted that a linear transformation has an inverse if and only if it is both into and onto. Let's recall our results about when a linear transformation from [mathjaxinline]\mathbb R^ n[/mathjaxinline] to [mathjaxinline]\mathbb R^ m[/mathjaxinline] can or must be into or onto.

Suppose [mathjaxinline]T: \mathbb R^ n \rightarrow \mathbb R^ m[/mathjaxinline] is a linear transformation.

If [mathjaxinline]n<m[/mathjaxinline], what can we conclude about [mathjaxinline]T[/mathjaxinline]?

If [mathjaxinline]n>m[/mathjaxinline], what can we conclude about [mathjaxinline]T[/mathjaxinline]?

Answer:

If [mathjaxinline]n<m[/mathjaxinline], our codomain is greater than our domain. Therefore, it may be true that [mathjaxinline]T[/mathjaxinline] maps any distinct [mathjaxinline]u[/mathjaxinline], [mathjaxinline]v[/mathjaxinline] [mathjaxinline]\in V[/mathjaxinline] to different points ([mathjaxinline]T(u) \neq T(v)[/mathjaxinline]); i.e., our codomain will accommodate this. So [mathjaxinline]T[/mathjaxinline] might be into. However, this is merely allowed, not required, by the circumstances. ([mathjaxinline]T[/mathjaxinline] could map everything to zero, for example.) So [mathjaxinline]T[/mathjaxinline] might or might not be into.

[mathjaxinline]T[/mathjaxinline] can only map each point of the domain to a single point in the codomain. We see that with [mathjaxinline]n<m[/mathjaxinline], the image of the domain can't equal the codomain. [mathjaxinline]T[/mathjaxinline] can't be onto.

If [mathjaxinline]n>m[/mathjaxinline], our domain is greater than our codomain. Every point in our domain must be mapped to something in the codomain by [mathjaxinline]T[/mathjaxinline]. Therefore we must have at least one pair of distinct points [mathjaxinline]u[/mathjaxinline], [mathjaxinline]v[/mathjaxinline] [mathjaxinline]\in V[/mathjaxinline] such that [mathjaxinline]T(u) = T(v)[/mathjaxinline]. [mathjaxinline]T[/mathjaxinline] can't be into.

Furthermore, if [mathjaxinline]n>m[/mathjaxinline], we have sufficient points in our domain to map to every point in the codomain, so the image of the domain could be the codomain. However, this is merely allowed by the circumstances, not required, so [mathjaxinline]T[/mathjaxinline] might or might not be onto.

Again, let [mathjaxinline]T: \mathbb R^ n \rightarrow \mathbb R^ m[/mathjaxinline] be a linear transformation. Given the above, if [mathjaxinline]n<m[/mathjaxinline], is it possible for [mathjaxinline]T[/mathjaxinline] to be invertible?

If [mathjaxinline]n>m[/mathjaxinline], is it possible for [mathjaxinline]T[/mathjaxinline] to be invertible?

Answer:

We've seen that, if [mathjaxinline]n<m[/mathjaxinline] or if [mathjaxinline]n>m[/mathjaxinline], T can't be both into and onto. Therefore, in neither case could T be invertible.

True or False: If a matrix [mathjaxinline]A[/mathjaxinline] is not square, it cannot be invertible.

True or False: If a matrix [mathjaxinline]A[/mathjaxinline] is square, it must be invertible.

Answer: Watch the last video again! Non-square matrices cannot be invertible, but not every square matrix is invertible. For instance, a square matrix of all zeros is not invertible.

Invertible Matrix Theorem. Suppose that [mathjaxinline]T: \mathbb R^ n \rightarrow \mathbb R^ n[/mathjaxinline] is a linear transformation with (square) standard matrix [mathjaxinline]A[/mathjaxinline]. Then the following conditions are all equivalent (i.e., either all of them are true, or all of them are false):

  1. [mathjaxinline]T[/mathjaxinline] is into.

  2. [mathjaxinline]\mathrm{Ker}(T) = \{ \mathbf{0}\}[/mathjaxinline].

  3. The columns of [mathjaxinline]A[/mathjaxinline] are linearly independent.

  4. [mathjaxinline]Ax = \mathbf{0}[/mathjaxinline] has only the trivial solution.

  5. [mathjaxinline]A[/mathjaxinline] row reduces to have a pivot in every column (has no free variables).

  6. [mathjaxinline]A[/mathjaxinline] row reduces to the identity matrix.

  7. [mathjaxinline]A[/mathjaxinline] row reduces to have a pivot in every row.

  8. For every [mathjaxinline]v\in \mathbb R^ n[/mathjaxinline], the equation [mathjaxinline]Ax = v[/mathjaxinline] is consistent.

  9. The columns of [mathjaxinline]A[/mathjaxinline] span [mathjaxinline]\mathbb R^ n[/mathjaxinline].

  10. [mathjaxinline]T[/mathjaxinline] is onto.

  11. [mathjaxinline]T[/mathjaxinline] is invertible.

  12. [mathjaxinline]A[/mathjaxinline] is invertible.

Note: There will be more equivalent conditions added to this list in the near future!

Suppose [mathjaxinline]A[/mathjaxinline] is an [mathjaxinline]m\times n[/mathjaxinline] matrix which has at least one free variable. Can we conclude that the columns of [mathjaxinline]A[/mathjaxinline] do not span [mathjaxinline]\mathbb R^ m[/mathjaxinline]?

Suppose [mathjaxinline]A[/mathjaxinline] is an [mathjaxinline]n\times n[/mathjaxinline] matrix which has at least one free variable. Can we conclude that the columns of [mathjaxinline]A[/mathjaxinline] do not span [mathjaxinline]\mathbb R^ n[/mathjaxinline]?

Answer: For the first question, we cannot make that conclusion, since we are not told the matrix is square. For instance, [mathjaxinline]\left[ \begin{array}{ccc} 1 & 0 & 0 \\ 0 & 1 & 0 \end{array} \right][/mathjaxinline] has a free variable, but its columns span [mathjaxinline]\mathbb R^2[/mathjaxinline]. For the second question, we know the matrix is square, so the Invertible Matrix Theorem applies. Since [mathjaxinline]A[/mathjaxinline] has a free variable, we can conclude that none of the conditions in the Invertible Matrix Theorem are true. In particular, its columns do not span [mathjaxinline]\mathbb R^ n[/mathjaxinline].

Let [mathjaxinline]A = \left[ \begin{array}{cc} 1 & 2 \\ 2 & 5 \end{array} \right][/mathjaxinline]. By row-reducing [mathjaxinline]A[/mathjaxinline], answer the question, is [mathjaxinline]A[/mathjaxinline] invertible?

Answer:

[mathjaxinline]A[/mathjaxinline] row reduces to the identity matrix, so it is invertible.
